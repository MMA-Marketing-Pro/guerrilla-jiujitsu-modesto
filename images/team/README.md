# Team Photos

Place instructor and staff headshots here.

**Naming convention:** `firstname-lastname.jpg`
**Recommended:** square crop, minimum 400×400px, JPEG or WebP.

Currently referenced:
- `mike-cruz.jpg` (Owner / Head Instructor — used on `about.html` and `index.html`)

When you drop a real photo in, update the `src` attribute in the HTML (or leave as-is if the `data-local` path already matches — Unsplash placeholder will swap automatically if you rename).

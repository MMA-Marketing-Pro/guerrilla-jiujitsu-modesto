# Logo

Place your logo here. Recommended: **SVG** or **PNG with transparent background**, minimum 400px wide.

Expected filename: `logo.png` (or `logo.svg`).

The navbar references `images/logo/logo.png`. If no file is present, the styled text fallback `GUERRILLA JJ` displays instead (via `onerror` on the `<img>` tag). Drop the file in and it will render automatically.
